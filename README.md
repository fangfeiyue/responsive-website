# 响应式网站

### 前言
现在已经进入移动互联网时代，用户很多时候都是在手机上浏览网站的，但是也不刨除用户会在PC端浏览网站，但是什么网站都直接做一套移动端页面一套PC端页面无形中增加了开发成本，尤其是对一些纯展示类的网站开发多套页面显示是不合适的，这个时候我们就可以运用响应式开发技术用一套代码适配多终端，响应式开发可以让用户体验和开发效率完美结合。

### 技术栈
- 媒体查询
- flex布局
